# 网课截图与 Markdown 笔记自动生成器 (ScreenshotAssistant)

这是一个基于 Python 的轻量级桌面自动化工具，专为网课学习者和技术讲座爱好者设计。它能自动监测屏幕选定区域的画面变化（如 PPT 翻页），实时捕获截图并自动生成对应的 Markdown 笔记文件。

## ✨ 核心功能

- **智能翻页检测**：利用图像哈希算法（aHash）比对画面差异，自动识别 PPT 翻页。
- **直观选区工具**：通过全屏蒙层交互，允许用户用鼠标精准框选需要监控的屏幕区域。
- **自动组织笔记**：为每个讲座创建独立文件夹，自动将截图以相对路径插入 Markdown 文件。
- **低干扰运行**：配置完成后在后台静默运行，支持通过全局快捷键安全退出。

## 🛠️ 环境要求

- **操作系统**：Windows (由于使用了 `keyboard` 和 `ImageGrab` 库，目前在 Windows 上表现最稳定)
- **Python 版本**：Python 3.8+

## 🚀 快速开始

### 1. 安装依赖
在终端或命令行中执行以下命令安装必要的库：
```bash
pip install opencv-python Pillow keyboard numpy
```

### 2. 运行程序
```bash
python ScreenshotAssistant.py
```

### 3. 使用流程
1. **初始化配置**：启动后，在弹出的窗口中输入“讲座名称”并选择“保存目录”。
2. **选择监控区域**：屏幕会变暗，请按住 **鼠标左键** 拖拽，框选出视频中播放 PPT 的具体区域。
3. **自动监控**：选区完成后，程序开始工作。每当检测到画面发生显著变化（翻页）时，程序会自动截图并更新 Markdown 笔记。
4. **手动补充**：你可以随时打开生成的 `.md` 文件（推荐使用 Typora 或 Obsidian），在自动插入的图片下方补充文字笔记。
5. **安全退出**：在任何时候按下 **F12** 键，程序将安全停止。

## ⚙️ 进阶参数微调

如果程序运行效果不如预期，可以在 `ScreenshotAssistant.py` 的 `__init__` 方法中调整以下参数：

| 参数 | 默认值 | 说明 |
| :--- | :--- | :--- |
| `diff_threshold` | `12` | **灵敏度**。值越小越灵敏。如果画面微动就触发，请调大。 |
| `cooldown` | `5` | **冷却时间（秒）**。防止 PPT 翻页动画导致连续重复截图。 |

## 🧠 技术原理

1. **均值哈希 (aHash)**：程序将截图缩小至 8x8 并灰度化，通过比较像素与平均值的关系生成 64 位指纹。
2. **汉明距离比对**：通过计算相邻两帧指纹的差异位数（汉明距离）来判断画面是否发生实质性变化。
3. **多线程架构**：GUI 界面与监控逻辑运行在不同线程中，确保选区工具灵敏且监控过程不阻塞。
