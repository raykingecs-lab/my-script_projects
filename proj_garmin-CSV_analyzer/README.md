# Garmin 跑步数据全能分析看板 (Garmin Full Analyzer)

这是一个专为 Garmin 跑者设计的纯本地数据分析工具。通过上传 Garmin Connect 导出的 CSV 文件，即可在浏览器中获得深度、直观的可视化报告。**无需上传云端，完美保护您的运动隐私。**

## 🚀 核心功能

### 1. 智能双模式识别
系统会自动根据上传的文件内容切换分析模式：
- **趋势概览 (Macro Mode)**：解析 `Activities.csv`，分析长期跑量、体能趋势与 PB 纪录。
- **单次计圈 (Micro Mode)**：解析 `Lap.csv`，深度剖析单次运动的配速稳健性与技术指标分布。

### 2. 趋势概览特性
- **全量时间过滤**：通过顶部日期选择器，实时同步更新 KPI、热力图、趋势图及 PB 墙。
- **运动频率热力图**：仿 GitHub 风格，直观展示年度/阶段性运动密度。
- **专业 PB 墙**：自动提取 5k、10k、半马、全马纪录，支持 `H:MM:SS` 专业计时格式。
- **有氧效果分布**：按 Garmin 标准 5 级强度聚合分析训练负荷。

### 3. 单次计圈特性
- **还原级交互设计**：完全还原原版深度分析布局，支持 **DataZoom 缩放滑块** 进行公里级微观下钻。
- **三维分布直方图**：步长、心率、配速的频数分布，一目了然看清表现的一致性。
- **智能明细表**：自动高亮统计行，清晰区分每公里分段与总计数据。

## 🛠️ 技术栈
- **核心框架**：HTML5 / Vanilla JS
- **UI 样式**：Tailwind CSS (响应式卡片布局)
- **数据解析**：[PapaParse](https://www.papaparse.com/) (高效、稳健的 CSV 解析)
- **图形渲染**：[Apache ECharts](https://echarts.apache.org/) (工业级交互图表)

## 📖 使用指南

1. **导出数据**：
   - **趋势数据**：在 Garmin Connect Web 端“活动”列表页面，点击右上角“导出 CSV”。
   - **计圈数据**：进入某次具体活动详情页，点击“计圈”选项卡，然后点击右侧“导出 CSV”。
2. **加载看板**：
   - 直接用浏览器打开 `garmin_full_analyzer.html`。
3. **上传分析**：
   - 点击“上传 CSV”按钮或直接将文件拖入页面。
   - 使用顶部的日期筛选器（趋势模式）或图表滑块（计圈模式）进行交互分析。
