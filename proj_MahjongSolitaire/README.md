## 🏃 项目简介

**上海马拉松：多层对对碰** 是一款基于 HTML5 Canvas 开发的高难度、现代运动风消除类网页游戏。本项目深度融合了经典“对对碰”消除规则与《羊了个羊》式的复杂多层堆叠逻辑，将上海马拉松（SHM）的竞技精神转化为一场极具压迫感的智力竞赛。

在 **8 分钟** 的关门时间内，你必须消除屏幕上密密麻麻的 **120 个** 运动装备元素，推过 42.195km 的终点线！

## ✨ 核心特性

- **极致堆叠难度**：120 个元素高密度聚拢，产生 6-10 层的窒息堆叠感，考验极致的观察力。
- **写实装备绘制**：所有图标（竞速厚底跑鞋、大檐空顶帽、能量胶、赛道钟等）均通过 Canvas API 精细绘制，极高辨识度。
- **实时里程联动**：每一次成功的对碰消除都会转化为已跑里程，背景线路图随进度动态高亮。
- **竞技仪表盘**：模拟专业运动手表界面，实时追踪配速（倒计时）与完赛目标。
- **零依赖单文件**：纯 HTML/CSS/JS 实现，无外部图片或库，极速加载。

## 🎮 快速开始

### 1. 运行方式
无需配置任何环境，直接在现代浏览器中打开以下文件：
```bash
proj_MahjongSolitaire/MahjongSolitaire.html
```

### 2. 操作指南
- **寻找激活图标**：只有最上层（不被任何其他物体挡住）的图标才是亮色可点击的。
- **匹配消除**：点击两个相同的激活图标即可消除。
- **关门挑战**：请在 420 秒（8 分钟）内完成所有匹配，否则将面临 DNF（未完赛）。
