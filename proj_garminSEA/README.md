# Garmin 跑步数据分析工具 (GarminSEA)

GarminSEA 是一个专为 Garmin 跑者设计的轻量级、零配置数据分析工具。它旨在解决 Garmin Connect 原始 CSV 导出的计圈数据不够直观的问题，通过本地浏览器解析，提供明细表格与核心运动指标的频数分布分析。


## 🌟 核心功能

- **智能数据解析**：自动识别 Garmin 导出的 CSV 文件中的“计圈行”与“总结行”。
- **精简视觉体验**：v2.4 版本舍弃了 0.1s 的配速小数位，使界面更加清爽、易读。
- **三大核心分布趋势图**：
  - **平均步长分布 (m)**：衡量步幅的稳定性。
  - **平均心率分布 (bpm)**：识别有氧/无氧耐力的集中区间。
  - **平均配速分布 (MM:SS)**：直观展现配速控制的精准度。
- **交互式可视化**：图表支持鼠标滚动缩放 (Inside) 和底部滑块缩放 (Slider)，轻松应对超过 100 圈的长距离数据。
- **响应式布局**：针对宽屏优化，确保在大屏幕上具有极佳的阅读舒适度。

## 📊 数据处理规范 (Standardized Schema)

工具从 CSV 中提取并标准化以下 8 个核心字段：

| 网页显示字段名 | 原始 CSV 字段名 | 特殊处理逻辑 |
| :--- | :--- | :--- |
| **计圈** | `计圈` | 识别 Lap（计圈）与 Summary（总结）行。 |
| **累计时间** | `累积时间` | 自动去除小数点后的毫秒部分。 |
| **平均配速** | `平均配速分钟/千米` | 统一格式化为 `MM:SS`。 |
| **平均心率** | `平均心率bpm` | 转换为整数。 |
| **最大心率** | `最大心率bpm` | 转换为整数。 |
| **累计爬升** | `累计爬升米` | 保留原始数值。 |
| **平均步频** | `平均步频每分钟步数` | 直接映射。 |
| **平均步长** | `平均步长米` | 转换为两位小数。 |

## 🛠️ 技术栈

- **[PapaParse v5.3.2](https://www.papaparse.com/)**：用于高性能、可靠的 CSV 解析。
- **[ECharts v5.4.3](https://echarts.apache.org/)**：强大的数据可视化引擎。
- **Vanilla JS & CSS**：无需安装任何依赖，极致的加载速度。

## 🚀 使用指南

1. **导出数据**：在 [Garmin Connect](https://connect.garmin.com/) 的活动计圈页面，点击“导出 CSV”。
2. **打开工具**：在浏览器中直接打开 `garmin_analyzer.html` 文件。
3. **上传分析**：
   - 点击“选择 CSV 文件”或拖入文件。
   - 页面将自动生成数据明细表。
   - 向下滚动查看心率、步长、配速的分布趋势图。

## 📦 本地运行

该项目是一个纯静态的 HTML 页面，无需任何后端环境：

```bash
# 克隆仓库
git clone <repository-url>

# 进入目录
cd proj_garminSEA

# 直接打开 html 文件即可
open garmin_analyzer.html
```
