# 🚀 HTML to Obsidian Batch Converter

这是一个专门为 **Obsidian** 用户打造的专业级 HTML 批量转换工具。它可以将本地导出的 HTML 笔记（如为知笔记 WizNote、印象笔记等）深度清洗并转换为纯净、排版精美、附件链接完好的 Markdown 文件。

---

## 🌟 核心特性

### 1. 深度内容清洗 (Deep Cleaning)
*   **彻底剔除噪声**：自动物理删除 `<style>`, `<script>`, `<head>`, `<nav>`, `<header>`, `<footer>` 等 12 类干扰标签。
*   **页眉残余过滤**：内置强大的正则表达式，自动识别并剔除笔记导出时常见的“HTML / 日期 / 星期”等导航文字。
*   **空元素净化**：递归清理所有无实质内容的嵌套 `div`, `span`, `p` 标签，解决 Markdown 中出现多余点号/横杠的问题。

### 2. 完美附件链路 (Rock-solid Attachments)
*   **占位符避让技术**：独创占位符替换机制，彻底解决 `markdownify` 转换时自动给图片文件名（含空格、括号、下划线）添加反斜杠转义导致的链接失效问题。
*   **智能搜索与重命名**：支持 URL 解码及资源文件夹（`_files`, `.assets`）模糊匹配，并采用 `{HTML名}_{图片名}` 唯一命名法防止覆盖。
*   **Obsidian 规范**：生成标准的 `![[attachments/filename]]` 格式，支持全库自动索引。

### 3. 极佳的用户体验
*   **图形化配置**：集成 Tkinter GUI 界面，点击“浏览”即可选择路径，无需修改任何代码。
*   **多线程并发**：核心逻辑在后台异步运行，处理万级文件时界面依然流畅。
*   **实时进度监控**：控制台提供 `tqdm` 进度条，实时显示百分比、速率及预计剩余时间。
*   **编码全兼容**：自动识别处理 UTF-8, UTF-16, GBK 等多种编码，告别 Windows 导出的 `0xff` 乱码。

## 🛠️ 运行准备

### 1. 环境要求
*   **Python**: 3.10 或更高版本
*   **系统**: Windows / macOS / Linux

### 2. 安装依赖包
```bash
pip install beautifulsoup4 markdownify lxml tqdm
```

---

## 📖 使用指南

1.  **启动程序**：
    ```bash
    python html2md.py
    ```
2.  **配置路径**：在弹出的窗口中，先后选择您的 **HTML 源文件夹** 和 **Markdown 输出文件夹**。
3.  **开始转换**：点击“开始转换”按钮，并在控制台查看详细进度。
4.  **导入 Obsidian**：转换完成后，将输出目录直接移动到您的 Obsidian 库（Vault）中即可。

---

## 📄 技术规格
*   **标题格式**: ATX (`# `)
*   **列表格式**: `- `
*   **编码模式**: 二进制读取 + 多级解码回退
*   **清洗深度**: DOM 级 + 正则级双重过滤
